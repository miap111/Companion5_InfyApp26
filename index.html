🌿 Theme Implementation

  • Earthy color palette + accents: We used greens and browns to match the land and pollution theme, while blue and yellow accent colors highlight important buttons, interactions, and gameplay moments.
  • Consistent font choices: A bubbly title font creates a fun and inviting atmosphere while the readable rounded body font keeps everything accessible without losing the playful vibe.
  • Simple layout + clear navigation: The experience moves smoothly from title screen to gameplay, keeping interactions easy and intuitive.
  • Pop-up task windows: Tasks appear in centered windows with clear instructions and easy navigation options.
  • Minimal gameplay UI: Interface elements stay tucked into corners so exploration and the environment remain the main focus.
  • Accessible buttons: Large rounded buttons make interactions simple, readable, and touch friendly.
