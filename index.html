Talking Clean

Making environmental education fun, playful, and interactive.

🎯 Pitch

Talking Clean is an interactive educational experience that teaches users about sustainability and environmental awareness through playful design, engaging characters, and game-inspired interactions.

🌿 Theme Implementation
  • Earthy palette with playful accents.
  • Bubbly and readable font system.
  • Simple game-like navigation flow.
  • Pop-up task UI design.
  • Minimal clutter gameplay focus.
🧠 Why We Built This

We wanted to create a project that makes learning about sustainability feel approachable and enjoyable for younger audiences while still encouraging meaningful environmental awareness.
