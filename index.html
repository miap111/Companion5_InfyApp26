🌿 Theme Implementation

  • Earthy color palette + accents: We used greens and browns to match the land/pollution theme, with blue and yellow accent colors to create contrast and highlight important buttons and features.
  • Consistent font choices: A bubbly title font makes the game feel fun and inviting, while a readable bubbly font keeps the rest of the text clear but still matches the vibe.
  • Simple layout + clear navigation: The game starts with a title screen and play button, then transitions smoothly into gameplay.
  • Pop-up task windows: Tasks appear inside centered pop-up windows with clear instructions and easy options.
  • Minimal UI during gameplay: UI elements stay tucked into corners so gameplay remains the main focus.
  • Accessible button design: Buttons are large, rounded, and easy to tap.
